# Public API Manifest

This manifest is the stable public API allowlist for this skill. It is intended
for API selection and symbol gating when generating lua-eco code.

Rules:
- Keep module and symbol names stable and source-location independent.
- Do not include source filenames, line numbers, or C loader symbols.
- Use references/01-07 for API selection guidance and behavior notes.
- Run scripts/audit-public-api.sh after changing public APIs or source docs.
